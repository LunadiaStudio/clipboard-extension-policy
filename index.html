Privacy Policy

This Chrome Extension, Clipboard Extension, is designed to help users manage and reuse their clipboard history efficiently. It respects your privacy and does not collect any personal or identifiable information.

Data Collection

  • No personal data is collected.
  • No data is transmitted to external servers.
  • All clipboard data is stored locally in your browser via chrome.storage.local.

Permissions Used

  • clipboardRead: To read clipboard content (text or images) when the popup is open.
  • clipboardWrite: To allow users to re-copy previously saved clipboard items.
  • storage: To save clipboard items and user settings locally.

Security

We do not use any remote code or third-party analytics. All code runs locally and does not communicate with outside sources.
